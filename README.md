# Backpack Pal

> **Status:** 🚧 In Progress

A full-stack web application designed to help people manage their backpacking trips with ease.

## ✨ Features
- **Trip Planner**
- **Inventory System**

## 🛠️ Tech Stack 
**Frontend:**
- Next.js (App Router)
- TypeScript
- Tailwind CSS & shadcn/ui
- Hosted on Vercel

**Backend:**
- FastAPI (Python)
- PostgreSQL
- SQLAlchemy + Alembic (Migrations)
- Hosted on Render

**Architecture:**
- Domain-driven modular design
- JWT Auth
