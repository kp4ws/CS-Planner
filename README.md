# 🎓 CS Degree Planner

> **Status:** 🚧 In Progress

A full-stack web application designed to help Computer Science students smoothly plan and track their degree progression. This project serves as the foundation for a larger, comprehensive student hub that will eventually expand to include job tracking, assignment tracking, and more.

## ✨ Features
- **Secure Authentication:** JWT-based authentication with secure bcrypt password hashing.
- **Account Management:** Protected routing and user profile management.
- **Degree Tracking:** Comprehensive planner with course, credit, and semester tracking.
- **Smart Audit Parsing:** PDF degree audit parsing for automatic degree plan generation *(Planned)*.

## 🛠️ Tech Stack 
**Frontend:**
- Next.js (App Router)
- TypeScript
- Tailwind CSS & shadcn/ui
- Hosted on Vercel

**Backend:**
- FastAPI (Python)
- PostgreSQL
- SQLAlchemy + Alembic (Migrations)
- Hosted on Render

**Architecture:**
- Domain-driven modular design
- JWT Auth
